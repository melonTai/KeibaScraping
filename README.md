# KeibaScraping
スクレイピングによりレースや馬データを収集する

# 使用方法
## scrape_results.py
過去のレース結果を取得し、年ごとにcsvで取得

同スクリプトで取得済みcsvのアップデートも可能

'''
python scrape_results.py [レース場id] [何年から] [何年まで]
'''
